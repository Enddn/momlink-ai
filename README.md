# MOM-LINK AI

### 외국인·다문화 임산부를 위한 AI 임신·출산 정보격차 해소 서비스

> K-AI Contents Award · Track B(솔루션 부문 — 일상의 정보 격차 해소) 출품용 MVP

MOM-LINK AI는 한국어와 의료·행정 용어가 익숙하지 않은 외국인·다문화 임산부가
임신 주수와 거주 지역에 따라 필요한 검사, 병원 방문 준비, 보건소 지원, 다문화가족 지원 정보를
**쉽게 이해하고 실제 행동으로 옮길 수 있도록** 돕는 AI 기반 임신·출산 안내 서비스입니다.

---

## 1. 해결하고자 하는 문제

한국에서 임신·출산을 준비하는 외국인·다문화 임산부는 다음과 같은 정보 격차를 겪습니다.

- 한국어 의료 용어 이해의 어려움
- 임신 주수별로 무엇을 해야 하는지 정보 부족
- 보건소·정부 지원 제도 탐색의 어려움
- 병원 방문 시 의사소통 부담
- 거주 지역별 산부인과·보건소·다문화가족센터 정보 접근성 부족

기존 임신·육아 앱은 한국어 사용자 중심이며 콘텐츠·커뮤니티·쇼핑에 강점이 있습니다.
MOM-LINK AI는 그 대신 **"내 상황에 맞는, 이해 가능한 안내"** 와 **"실제 행동으로 이어지는 도움"** 에 집중합니다.

> "저는 한국어가 서툰 임산부입니다. 지금 임신 18주차이고, 경기도에 살고 있어요.
> 이번 시기에 어떤 검사를 받아야 하고, 보건소에는 무엇을 문의해야 하나요?"

---

## 2. AI 사회문제 해결 관점 (공모전 연계)

- **정보 격차 해소**: 의료·행정 용어를 쉬운 한국어 + 모국어 안내로 변환
- **실생활 문제 해결**: 병원 질문 카드, 보건소 문의 문장처럼 바로 쓰는 산출물 제공
- **취약 계층 지원**: 언어 장벽이 있는 외국인·다문화 임산부를 우선 대상으로 설계
- **안전 우선 설계**: 진단·처방을 하지 않고, 항상 담당 의료진 확인을 안내

---

## 3. 서비스 대상

**주요 사용자**
- 한국어가 익숙하지 않은 외국인 임산부
- 결혼이민자 및 다문화가족 임산부
- 한국 의료·복지 제도 이용이 어려운 임산부
- 병원 방문 전 질문을 정리하고 싶은 초산 임산부

**확장 사용자**
- 다문화가족지원센터 상담사 / 보건소 임산부 지원 담당자
- 외국인 주민 지원 기관 / 산부인과 및 지역 의료기관

---

## 4. 주요 기능

| 화면 | 기능 |
|---|---|
| **홈** | 프로필 요약 · 임신 주수별 맞춤 체크리스트(5구간 자동 전환) · 안전 배너 · 바로가기 |
| **쉬운 설명** | 의료·행정 용어를 쉬운 말로 풀고, 병원·보건소에 물어볼 문장 제시 · 검색 지원 |
| **질문 카드** | 병원 방문 질문 카드 / 보건소 문의 문장 자동 생성 · 복사 · 선택 언어별 안내 문구 |
| **기관 찾기** | 거주 지역 기반 mock 기관 목록 · 유형 필터 · 데모용 정보 명시 |
| **AI 상담** | rule-based mock 챗봇(임당·보건소·병원·영양제·응급 분기) · 안전 문구 항상 포함 |

### 4.1 임신 주수별 맞춤 체크리스트
입력한 주수에 따라 1~12 / 13~20 / 21~28 / 29~36 / 37~40주 구간별 항목을 제공합니다.

### 4.2 쉬운 언어 변환
| 어려운 표현 | 쉬운 설명 |
|---|---|
| 임신성 당뇨 검사 | 임신 중 혈당이 높은지 확인하는 검사 |
| 기형아 선별검사 | 아기의 건강 상태를 확인하기 위한 검사 |
| 보건소 임산부 등록 | 보건소에 임산부 정보를 등록하고 지원을 받는 절차 |

### 4.3 병원 질문 카드 / 보건소 문의 문장
주수·지역·언어를 반영해 자동 생성하고, 복사 후 병원·보건소에서 그대로 보여줄 수 있습니다.
선택 언어(한·영·중·베트남·일·몽골·태국·러)별 안내 문구를 함께 표시합니다(현재 mock translation).

## 5. 기술 스택

- **Next.js 14 (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **lucide-react** (아이콘)
- 모바일 우선 반응형 · 클라이언트 상태 + `localStorage` 기반
- 로그인 / 실제 의료 진단·처방 기능 없음

## 8. MVP 범위

**포함**
- 온보딩 입력 + `localStorage` 저장 (재접속 시 홈부터)
- 주수별 체크리스트, 쉬운 설명, 질문 카드/문장 생성, 복사
- 지역 기반 mock 기관 목록, rule-based mock 챗봇, 안전 문구

**제외 (향후 단계)**
- 실제 번역 API / LLM API 연동
- 실제 병원·보건소·정부 지원 데이터 연동
- 로그인·회원 관리, 의료 진단·처방

---

## 9. 의료 안전 안내

이 서비스는 의료 진단·처방·응급 판단을 제공하지 않습니다.

- "진단합니다", "복용하세요", "괜찮습니다" 같은 단정 표현을 사용하지 않습니다.
- 항상 "담당 의사 또는 병원에 확인하세요"로 안내합니다.
- 응급 증상 시 "즉시 119 또는 가까운 응급실에 문의하세요"로 안내합니다.

> MOM-LINK AI는 의료 진단이나 처방을 제공하지 않습니다. 건강 상태와 검사 여부는 반드시
> 담당 의료진에게 확인해주세요. 갑작스러운 출혈, 심한 복통, 호흡곤란 등 응급 증상이 있으면
> 즉시 119 또는 가까운 응급실에 문의하세요.

## 10. 향후 확장 방향

- 실제 번역 API 연동으로 8개 언어 전체 콘텐츠 번역
- LLM 기반 상담으로 rule-based 챗봇 고도화 (안전 가드레일 유지)
- 공공데이터 연동: 지역 보건소·다문화가족센터·산부인과 실데이터
- 정부 임산부 지원 정책 자동 매칭 (지역·소득·체류자격 기반)
- 보건소·다문화센터 상담사용 관리자 뷰
